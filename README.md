# Agent-Based Artificial Intelligence - Hands-on Lessons Codes (A.Y. 2025/2026)

This repository contains the material developed during the hands-on lessons of the **Agent-Based Artificial Intelligence** course.

At the moment, the repository includes an initial **Python** implementation focused on modeling a graph search problem.

## Currently Available Contents

At the moment, the repository contains a first implementation of a **tree search algorithm**, together with the supporting classes and a few simple usage examples.
